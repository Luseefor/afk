## Provider registration example

```python
from afk.llms import register_llm_provider, list_llm_providers

register_llm_provider(my_provider)
print(list_llm_providers())
```

## Adapter capability map

| Capability | Meaning |
| --- | --- |
| streaming | supports stream events |
| structured_output | supports schema-based outputs |
| tools | supports tool-calling contract |
| idempotency | supports idempotency semantics |
