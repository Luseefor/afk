title: Failure Policy Matrix
description: Retry and terminal-failure behavior across AFK modules.
---

| Subsystem | Retryable | Terminal | Notes |
| --- | --- | --- | --- |
| LLM transport timeout | Yes | No | bounded by retry policy |
| Invalid LLM payload | No | Yes | validation failure |
| Unknown queue contract | No | Yes | cannot route safely |
| Optional subagent failure | No | No | degrade per join policy |
| Telemetry export error | No | No | non-fatal instrumentation |
